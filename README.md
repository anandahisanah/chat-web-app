# Getting Started

Simple Chat Web App using Ruby on Rails API for Back End, VueJS and TailwindCSS for Front End

## Running Test
Need to run migration first. Then, `bin/rails test`

## Back End
Run migration

`bin/rails db:migrate`

Run back end 

`rails s`

## Front End
Move directory

`cd frontend/`

Run front end

`npm install`

`npm run dev`
